# Catálogo inicial de amenazas

## 1. Objetivo

Identificar amenazas relevantes para los activos del laboratorio de AlmeriaLogiTech S.L.

---

## 2. Amenazas identificadas

| ID Amenaza | Amenaza | Descripción |
|---|---|---|
| T-001 | Acceso no autorizado | Uso indebido de credenciales o bypass de autenticación |
| T-002 | Inyección SQL | Manipulación de consultas contra la base de datos |
| T-003 | XSS | Ejecución de JavaScript en navegador de usuario |
| T-004 | Robo de sesión | Captura o reutilización de cookies de sesión |
| T-005 | Movimiento lateral | Desplazamiento entre redes tras compromiso inicial |
| T-006 | Exposición de servicios internos | Servicios críticos accesibles desde redes no necesarias |
| T-007 | Enumeración de Active Directory | Obtención de información sobre usuarios, grupos y servicios |
| T-008 | Fuerza bruta / password spraying | Ataques contra cuentas de usuario |
| T-009 | Exfiltración de datos | Extracción no autorizada de información |
| T-010 | Pérdida de datos | Pérdida por ausencia de backup o restauración |
| T-011 | Interceptación de comunicaciones | Captura de tráfico no cifrado |
| T-012 | Explotación de servidor web | Compromiso por mala configuración o vulnerabilidad |
| T-013 | Abuso de privilegios | Uso de permisos excesivos para ampliar impacto |
| T-014 | Divulgación de información | Exposición de versiones, rutas o estructura interna |
| T-015 | Falta de detección | Incapacidad para identificar eventos de seguridad |
